# 方針
- 無難かつ(ライブラリ的には)複雑でないナンカイイカンジの構成にする
  - ライブラリなどは可能な限りメジャーかつシンプルな形に留める
  - 作り方やプログラムは好き放題やる

# 採用ライブラリ
- cysharp.unitask
  - 非同期処理用のライブラリ。
  - ないと辛いし使ってみたいとのことだったし採用。他に代用知らないかも。
- cysharp.r3, observablecollections.r3
  - メッセージングライブラリ。
  - 何かしらはないと辛く、一番メジャーだし採用
- DoTween
  - アニメーション、トゥウィーンライブラリ。
  - 8割ぐらいで使うので入れとく
- UIEffects
  - UI Image用のエフェクト詰め合わせ。
  - 8割ぐらいで使うので入れとく
